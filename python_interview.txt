Shuffling of list elements in python
====================================
from random import shuffle
x = [1,2,3,4,'sai','raj']
shuffle(x)
print(x)


sorting of list
===============

l1=[1,7,3,6,2,5,3,99,67,77]

l1 = ['1','2','89','32','56','12']
